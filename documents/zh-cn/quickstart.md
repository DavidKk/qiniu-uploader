# 开速开始

## 安装

```Javascript
npm install qiniup
```

## 使用

```Javascript
import { Uploader } from 'qiniup'

let qiniup = new Uploader()

let params = { token: 'generated by sdk on server' }
qiniup.upload(file, params, function (error) {
  if (error) {
    console.error(error)
    return
  }

  console.log('upload success')
})
```
